Lab 8 - Anon Functions and Callbacks

Challenge

Experiment with anonymous functions and callbacks in the console as well as in a javascript file.

Problems

We had a lot of problems trying to get the prompt() function to work along side with printing the results of that on the webpage. We eventually found out that we needed to use document.open() and move the script statement towards the bottom so it would print there instead of replacing the entire webpage.

Reflection

We were able to do something funny for the functions we created and everything worked out for us very well. Aside from the text input issues, we were able to learn and understand more about callbacks and anonymous functions through this lab.
